# AI Solution Design for Agriculture

## Domain
Agriculture

## Project Topic
Crop Disease Detection using AI

## Project Overview
This project designs an AI-based solution for detecting crop diseases using leaf images. The system uses Computer Vision and a CNN model to identify whether a crop leaf is healthy or diseased.

## AI Technique Used
Computer Vision with Convolutional Neural Network (CNN)
